# DEM
The Digital Ecosystem Model is an ontology for modelling digital ecosystems, developed by the PERICLES project.

It will be released in Q4 2016.

The [EcoBuilder] (https://github.com/pericles-project/ecobuilder) tool will facilitate the simple creation of Digital Ecosystem Models for user scenarios.
